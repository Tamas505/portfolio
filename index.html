Rólam

56 évesen döntöttem úgy, hogy új szakmát tanulok, és a webfejlesztésben találtam meg azt a területet, ahol igazán otthon érzem magam. Erősségeim a kitartás, a precizitás, és az, hogy szeretek mélyen megérteni minden technológiát, amivel dolgozom, nem csak használni őket. Jelenleg főként a frontendre fókuszálok (HTML, CSS, Bootstrap, JavaScript, TypeScript, Angular), a képzésben pedig hamarosan a backend fejlesztés következik. Célom, hogy junior fejlesztőként egy támogató csapatban dolgozhassak, ahol értéket teremthetek és folyamatosan fejlődhetek.

Technológiák, amikkel dolgozom

  • HTML5, szemantikus struktúra
  • Modern CSS (flexbox, grid, reszponzív layoutok)
  • Bootstrap
  • Alap JavaScript, TypeScript
  • Angular – komponensek, űrlapkezelés, validáció
  • Git és GitHub, VS Code
  • Folytatólagosan Backend tanulás
